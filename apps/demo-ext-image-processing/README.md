# demo-ext-image-processing

Demo app for the [`@niivue/nv-ext-image-processing`](../../packages/nv-ext-image-processing) extension — Otsu thresholding, haze removal, and more.

## Getting Started

```bash
bun install                          # From monorepo root
bunx nx dev demo-ext-image-processing # Start dev server
```

## Build

```bash
bunx nx build demo-ext-image-processing
```
