# demo-ext-save-html

Demo app for the [`@niivue/nv-ext-save-html`](../../packages/nv-ext-save-html) extension — export a NiiVue scene as a self-contained HTML file.

## Getting Started

```bash
bun install                    # From monorepo root
bunx nx dev demo-ext-save-html # Start dev server
```

## Build

```bash
bunx nx build demo-ext-save-html
```
