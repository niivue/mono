# demo-ext-drawing

Demo app for the [`@niivue/nv-ext-drawing`](../../packages/nv-ext-drawing) extension — drawing interpolation and segmentation tools.

## Getting Started

```bash
bun install              # From monorepo root
bunx nx dev demo-ext-drawing  # Start dev server
```

## Build

```bash
bunx nx build demo-ext-drawing
```
